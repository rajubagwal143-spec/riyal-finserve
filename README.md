# Riyal FinServe

Riyal FinServe is a simple loan application website.

## Features
- User loan application form
- Admin login panel
- Admin can view and update loan status
- Clean and responsive UI

## Tech Stack
- HTML
- CSS
- PHP
- MySQL

## How to Use
1. User fills loan application form
2. Admin logs in to dashboard
3. Admin approves or rejects loan requests
